# Entrega Etapa 2

Arquivos principais:
- `Proposta_Solucao_Analitica_Etapa2_v2.docx`
- `eda_scr_2025_v2.py`

Estrutura complementar:
- `eda_outputs/` -> saídas tabulares e resumo da análise
- `graficos/` -> gráficos utilizados no documento

## Como subir no GitHub
1. Extraia este pacote em uma pasta do seu projeto.
2. Coloque os arquivos CSV de 2025 na mesma pasta do script `eda_scr_2025_v2.py`.
3. Execute o script:
   `python eda_scr_2025_v2.py`
4. Faça o commit no repositório.

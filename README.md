## Organização dos arquivos
- `DOC/documentos_principais/`: documento principal da entrega.
- `DOC/materiais_auxiliares/graficos_eda/`: gráficos da análise exploratória.
- `DOC/materiais_auxiliares/tabelas_eda/`: tabelas e resumos gerados na EDA.
- `CODE/dados_brutos/`: arquivos CSV mensais do dataset, divididos em partes.
- `CODE/scripts/`: scripts Python da análise.

## Dataset
Os dados analisados são públicos, agregados e atualizados mensalmente pelo Banco Central, com informações sobre carteira ativa, inadimplência, ativo problemático, modalidade, segmento, UF e tipo de cliente.
